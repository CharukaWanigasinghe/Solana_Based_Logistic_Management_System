💾 Hybrid Storage Model

ON-CHAIN (Solana Blockchain)

  • ✓ Event hashes for integrity verification
  • ✓ Event type classification
  • ✓ Immutable timestamps
  • ✓ Shipment status changes
  • ✓ Delivery confirmation records
  • ✓ Complete audit trail

OFF-CHAIN (IPFS/Arweave Network)

  • • Detailed GPS sequences & paths
  • • Temperature sensor readings
  • • Humidity measurements
  • • High-resolution delivery photos
  • • Recipient signatures
  • • Additional documentation

🔗 Why Hybrid? Critical shipment data stays on-chain for security and immutability, while detailed sensor/image data is stored off-chain for efficiency and cost-effectiveness, with cryptographic links ensuring integrity.
