Complete System Guide & User Manual

📋 System Overview

INTELLICA is a comprehensive blockchain-powered logistics management platform that combines traditional web technologies with cutting-edge Web 3.0 blockchain features. The system enables secure, transparent, and immutable tracking of shipments from creation to delivery using the Solana blockchain.

Key Innovation: This system bridges traditional logistics with blockchain technology, providing immutable records, cryptographic security, and decentralized trust for supply chain management.

🏗️ How the System Works

Architecture Flow

User Browser → FastAPI Backend → Solana Blockchain → Smart Contract ↓ ↓ ↓ ↓ HTML/JS Python Logic RPC Calls Rust Program Forms Data Processing Transactions State Changes Maps Session Mgmt Verification PDA Accounts

Core Process Flow

User Authentication
Users connect via Phantom Wallet (Solana's browser extension). Wallet address serves as user identity - no passwords needed. Session stored securely in browser cookies.
Shipment Creation
User enters shipment details (ID, destination, product, quantity, GPS). System creates blockchain transaction. Data stored on Solana blockchain (immutable, timestamped).
Tracking Events
GPS coordinates logged at each checkpoint. Environmental data (temperature, humidity) recorded. Each event becomes a blockchain transaction with complete audit trail.
Delivery Confirmation
Recipient signature captured. Delivery photo hash stored. Final blockchain confirmation creates permanent proof of delivery.
History & Verification
Complete shipment journey viewable on interactive map. GPS data integrity verification with cryptographic proof of all events.

👥 How Users Use the System

For Logistics Managers

Access the System
Open browser and navigate to http://localhost:8000. Click "Login" and connect your Phantom wallet. The system recognizes your wallet address as your user identity.
Create Shipments
Navigate to "Blockchain Tracking" section. Fill in shipment details (ID, destination, product line, quantity, GPS coordinates). Click "Create & Deploy to Blockchain". System redirects to confirmation page showing blockchain transaction details.
Monitor Shipments
Use "Tracking History" to view complete shipment journeys on interactive maps. See real-time status updates, environmental conditions, and access complete audit trails with timestamps.
Verify Data Integrity
Use GPS verification tools to confirm data authenticity. View cryptographic proofs of all transactions. Generate compliance reports for regulatory requirements.

For Delivery Drivers

Log Tracking Events
Enter shipment ID and current location details. Record GPS coordinates, temperature, and humidity readings. Click "Log Event On-Chain" for permanent blockchain recording.
Confirm Deliveries
Capture recipient signature and upload delivery photo (stored securely on IPFS). Click "Confirm Delivery" to create final blockchain record with proof of delivery.

For Customers/Recipients

Track Shipments
Access tracking history using shipment ID. View real-time location on interactive maps. Monitor delivery status and see estimated arrival times.
Verify Deliveries
Confirm delivery with digital signature. View immutable proof of delivery on blockchain. Access permanent delivery records for insurance or legal purposes.

🚀 Getting Started Guide

Prerequisites

🐍 Python 3.8+
Core runtime for backend services
📦 Node.js 16+
Frontend dependency management and CSS building
👛 Phantom Wallet
Solana blockchain wallet browser extension
📋 Git
Version control for code management

Installation Steps

Clone Repository
git clone <repository-url> cd "Solana Based Logistic Management System"
Setup Python Environment
python -m venv .venv # Windows: .venv\Scripts\activate # macOS/Linux: source .venv/bin/activate pip install -r requirements.txt
Setup Frontend (Optional)
npm install npm run build-css
Configure Environment
Create .env file with database and Solana RPC settings
Run Application
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
Access System
Open http://localhost:8000 in browser, connect Phantom wallet, and start using blockchain logistics features
